    Tổng quan Deployment

    Kiến trúc triển khai

    LegoCity có thể triển khai theo nhiều cách:

    ┌─────────────────────────────────────┐
│         Load Balancer / CDN         │
└──────────────┬──────────────────────┘
               │
    ┌──────────┴──────────┐
    │                     │
┌───▼──────┐      ┌───────▼────┐
│Dashboard │      │  Orion-LD  │
│(Next.js) │      │  (Broker)  │
└──────────┘      └────────────┘
    │                     │
    └─────────┬───────────┘
              │
         ┌────▼─────┐
         │ MongoDB  │
         └──────────┘

    Các nền tảng hỗ trợ

    1. AWS (Production Ready)

    • EC2 - Máy ảo
    • ECS/EKS - Container orchestration
    • RDS - MongoDB managed
    • CloudFront - CDN
    • Route 53 - DNS

    2. Cloudflare (Edge Computing)

    • Pages - Static hosting cho dashboard
    • Workers - Serverless functions
    • R2 - Object storage
    • Tunnels - Expose local services

    3. VPS / Máy ảo (Self-hosted)

    • Ubuntu/Debian server
    • Docker + Docker Compose
    • Nginx reverse proxy
    • Let's Encrypt SSL

    4. Coolify (PaaS)

    • One-click deployment
    • Tự động SSL
    • Built-in monitoring
    • Git-based deployment

    Yêu cầu hệ thống

    Tối thiểu

    • CPU: 2 cores
    • RAM: 4GB
    • Disk: 20GB SSD
    • Network: 100Mbps

    Khuyến nghị (Production)

    • CPU: 4+ cores
    • RAM: 8GB+
    • Disk: 50GB+ SSD
    • Network: 1Gbps
    • Backup: Daily automated

    Environment Variables

    Các biến môi trường cần thiết:

    # Database
DATABASE_URI=mongodb://localhost:27017/legocity

# Payload CMS
PAYLOAD_SECRET=your-secret-key-32-chars-min
NEXT_PUBLIC_SERVER_URL=https://yourdomain.com

# Orion-LD
ORION_URL=http://localhost:1026

# Optional: AI Integration
OPENAI_API_KEY=sk-...
ANTHROPIC_API_KEY=sk-ant-...

# Optional: Monitoring
SENTRY_DSN=https://...

    Checklist trước khi deploy

    • [ ] Đã cấu hình domain và DNS
    • [ ] Đã chuẩn bị SSL certificate
    • [ ] Đã backup dữ liệu (nếu migrate)
    • [ ] Đã test trên staging environment
    • [ ] Đã thiết lập monitoring
    • [ ] Đã cấu hình backup tự động
    • [ ] Đã document các API keys
    • [ ] Đã thiết lập rollback plan

    Security Best Practices

    1. Sử dụng HTTPS - Bắt buộc cho production
    2. Environment Variables - Không commit vào Git
    3. Firewall Rules - Chỉ mở ports cần thiết
    4. Regular Updates - Update dependencies định kỳ
    5. Rate Limiting - Chống DDoS và abuse
    6. Backup Strategy - Backup daily + retention policy
    7. Access Control - Phân quyền rõ ràng
    8. Monitoring - Alert khi có vấn đề

    Deployment Flow

    graph LR
    A[Code Push] --> B[CI/CD Pipeline]
    B --> C[Build & Test]
    C --> D{Tests Pass?}
    D -->|Yes| E[Build Docker Image]
    D -->|No| F[Notify Developers]
    E --> G[Push to Registry]
    G --> H[Deploy to Staging]
    H --> I{Manual Approval}
    I -->|Approved| J[Deploy to Production]
    I -->|Rejected| F
    J --> K[Health Check]
    K --> L{Healthy?}
    L -->|Yes| M[Success]
    L -->|No| N[Rollback]
