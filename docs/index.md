# Lego City Documentation

Welcome to the official documentation for **Lego City** - a modern Smart City Builder using FIWARE Enablers and NGSI-LD.

## What is Lego City?

Lego City is an open-source platform that simplifies the creation of smart city applications. It combines:

- **FIWARE Enablers**: Industry-standard smart city components
- **NGSI-LD**: Next Generation Service Interface for Linked Data
- **PayloadCMS**: Modern, customizable content management
- **Go Backend**: High-performance API server
- **Next.js Dashboard**: Beautiful, responsive web interface
